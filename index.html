About Me

My name is Diana Wallace and I am a Digital Illustrator and Web Developer currently based in Denver, Colorado

I am a dedicated, enthusiastic, and highly motivated digital illustrator and web developer with experience building web sites and creating illustrations. Formerly educated in fine art and illustration I have spent over seven years with the Adobe Suite of creative applications. I have a passion for art, design, and technology and enjoy blending these three mediums along with modern web technologies.

This site was built with cloud9 ide and bootstrap, which is an open source Javascript framework. The source code is available on Github.

Projects

Digital Illustration

Utilizing digital technologies of a wacom tablet and Photoshop and Illustrator I create illustrations of character, creature, and environment designs to depict narrative and compelling stories for both web and print media.

Responsive Web Design

I start building a website with mobile design first and then continue on to the larger screen formats. I design with color, typography and simple navigation and layout in mind to build a practical yet aesthetic experience for the user.

Frontend Development

I use HTML5, CSS3, jQuery, JavaScript, and bootstrap, which are the latest technologies to produce clean and semantic code. Whether it's a calculator, a shopping list, or a fun game the user is sure to enjoy my web and mobile apps.
